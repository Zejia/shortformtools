      Reading the number

      A better landed-cost workflow

      Most teams go wrong by stopping at the first clean-looking total. Use the three checks below before you call the number done.

      1. Audit the value basis

      If the quote is FOB but your model quietly used CIF, or the other way around, the landed number can look precise and still be structurally wrong.

      2. Compare a stressed file

      If a one-step freight slip or a two-point rate shift breaks the unit economics, the business decision should change before the PO does.

      3. Capture the assumptions

      Record the preset, rate, threshold mode, and basis used. That keeps Pricing, Ops, and Finance from quoting the same product from different math.

      Methodology

      How the page models landed cost

      The calculator is designed for planning, not filing. It separates customs value, duty, and tax logic so the assumptions stay readable when you share the result with a buyer, broker, or finance lead.

      1. Build the customs value base

      The file starts with product value and can include freight and insurance in the duty base depending on the declaration logic you are testing.

      2. Layer duty before tax where needed

      Duty is modeled separately so you can see when the tax base sits on top of customs value alone versus customs value plus duty and brokerage.

      3. Add the non-duty friction

      Brokerage, misc shipment costs, and the FX or compliance buffer are included because those line items still shape the real import cost per unit.

      Common questions

      Landed cost calculator FAQ

      What is included in landed cost?

      Landed cost usually means product value plus freight, insurance, duty, import tax or VAT, brokerage, and other shipment-level costs required to get goods cleared and ready to sell or receive.

      Why is landed cost different from import duty?

      Import duty is only one charge inside the shipment. Landed cost adds the other costs that still affect the final economics, which is why a tariff-only estimate can still understate the real per-unit number.

      When should I use the tariff-first calculator instead?

      Use the import duty calculator when the immediate question is rate treatment or customs value. Use this page when you need a quote-ready landed figure.

      Can threshold relief remove both duty and VAT?

      Sometimes, but not universally. The correct treatment depends on the market and collection method, which is why this page keeps relief mode adjustable instead of hiding it behind one generic checkbox.
