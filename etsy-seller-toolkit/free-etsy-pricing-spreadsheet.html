How to use it

  • Create one row for each listing or product variation.
  • Keep a normal-price row and a discounted row for each best seller.
  • Add a separate row for Offsite Ads if that channel affects the listing.
  • Use the monthly units column for a rough shop-level projection before changing prices.
  • Review shipping and packaging costs monthly, because small changes can erase margin.
