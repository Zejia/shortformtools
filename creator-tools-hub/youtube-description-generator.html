Description tips

Put the main promise or resource in the first two lines, then add context, links, and chapters. Avoid burying the most important link below a long intro. A description should help a viewer decide what the video covers, where to go next, and how to find the parts they care about.

The generator keeps the structure simple on purpose: summary, call to action, resource link, and chapter placeholders. That makes the output easy to edit for a tutorial, case study, Shorts compilation, podcast clip, or product walkthrough.
